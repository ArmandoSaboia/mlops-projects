# Conversational AI Assistant

## Technology Stack
- **LLM Framework**: LangChain
- **Models**: GPT-4
- **Vector DB**: Pinecone
- **Embeddings**: OpenAI
- **API Development**: FastAPI
- **Monitoring**: Prometheus
- **Cloud Platform**: Azure (OpenAI)
- **Data Validation**: Pydantic
