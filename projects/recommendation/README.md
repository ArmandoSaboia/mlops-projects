# Product Recommendation System

## Technology Stack
- **ML Framework**: PyTorch
- **Data Processing**: Apache Spark
- **Feature Store**: Feast
- **Model Serving**: TensorFlow Serving
- **Vector DB**: Pinecone
- **Stream Processing**: Apache Kafka
- **Monitoring**: ELK Stack
- **Cloud Platform**: Azure (AKS)
